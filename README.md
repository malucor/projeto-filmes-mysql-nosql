Este repositório contém um projeto da disciplina de **Introdução ao Armazenamento e Análise de Dados** da **Universidade Federal Rural de Pernambuco (UFRPE)**, dividido em duas atividades principais:

1. **Desenvolvimento de um sistema web CRUD** com interface gráfica em **Streamlit** conectado a um banco de dados **relacional MySQL**.  
2. **Apresentação prática (hands‑on)** de um banco de dados **NoSQL** (**Apache CouchDB**), com operações CRUD e comparações com o modelo relacional.
